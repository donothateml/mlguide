Random Forest

📌 Definition

Random Forest is an ensemble of multiple decision trees.

⚙️ How It Works

  • Create multiple trees
  • Each tree predicts
  • Final output = majority vote

🔢 Example

5 trees → outputs: Yes, Yes, No → Final = Yes

✅ Advantages

  • Reduces overfitting

❌ Disadvantages

  • Slower than single tree

🎯 Interview

Why use Random Forest?

Answer: Better accuracy.

🧪 Practice

Random forest uses how many trees?

Answer: Multiple
