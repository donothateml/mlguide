Support Vector Machine (SVM)

📌 Definition

SVM is a classification algorithm that finds the best boundary (hyperplane) between classes.

⚙️ How It Works

  • Find separating line
  • Maximize margin
  • Use support vectors

📐 Key Concept

Margin = Distance between boundary and nearest points

🔢 Example

Classify emails as Spam / Not Spam using boundary.

✅ Advantages

  • Works well for high-dimensional data

❌ Disadvantages

  • Slow for large datasets

🎯 Interview

What are support vectors?

Answer: Points closest to boundary.

🧪 Practice

SVM tries to maximize what?

Answer: Margin
