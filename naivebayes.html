Naive Bayes Algorithm

📌 Definition

Naive Bayes is a probabilistic classification algorithm based on Bayes’ Theorem with an assumption that features are independent.

📐 Formula

P(A|B) = P(B|A) * P(A) / P(B)

⚙️ How It Works

  • Calculate prior probability
  • Calculate likelihood
  • Apply Bayes theorem
  • Choose class with highest probability

🔢 Example

Email classification:

  • P(Spam|Offer) = high
  • P(Not Spam|Offer) = low

→ Classified as Spam

✅ Advantages

  • Fast and simple
  • Works well with text data

❌ Disadvantages

  • Assumes feature independence
  • Not accurate for correlated features

🎯 Interview Questions

Q: Why is it called “Naive”?

A: Because it assumes independence between features.

🧪 Practice

Naive Bayes is based on which theorem?

Answer: Bayes Theorem
