Decision Tree

📌 Definition

A Decision Tree is a supervised learning algorithm that splits data based on conditions.

⚙️ How It Works

  • Select best feature
  • Split data
  • Repeat recursively

📐 Key Concept

Information Gain = Entropy(before) - Entropy(after)

🔢 Example

Weather → Play?

  • Sunny → No
  • Rain → Yes

✅ Advantages

  • Easy to understand
  • No data scaling required

❌ Disadvantages

  • Overfitting problem

🎯 Interview

What is overfitting?

Answer: Model memorizes training data.

🧪 Practice

Decision trees use which concept?

Answer: Information Gain
