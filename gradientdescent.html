Gradient Descent

📌 Definition

Gradient Descent is an optimization algorithm used to minimize loss.

📐 Formula

θ = θ - η∇J(θ)

⚙️ How It Works

  • Start with random weights
  • Calculate gradient
  • Update weights
  • Repeat until convergence

🔢 Example

If θ = 5 and gradient = 2:

θ = 5 - 0.1×2 = 4.8

✅ Advantages

  • Simple and efficient

❌ Disadvantages

  • May get stuck in local minima

🎯 Interview

What is learning rate?

Answer: Step size of update.

🧪 Practice

Gradient descent minimizes what?

Answer: Loss function
